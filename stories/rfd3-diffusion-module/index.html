main scaffold

Atom14 slots -> atom pair-biased attention -> token pair refinement -> atom decoder.

RFD3 enters the diffusion module with dense atom14 slots, initializer outputs, and the current noisy coordinates. The model moves through atom-level attention, token-level refinement, and atom-level decoding.

pair-bias boundary

RFD3 uses explicit pair bias at both atom and token scale.

Atom attention consumes `P_LL`; token attention consumes `Z_II`. The atom pair bias is not simply noisy atom-distance bias. Noisy coordinates affect `Q_L`, local neighborhoods, and token distograms, while `P_LL` is built in the initializer.

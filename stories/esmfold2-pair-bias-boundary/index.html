bird's-eye scaffold

Conditioning -> atom encoder -> token transformer -> atom decoder.

`DiffusionModule.forward` first prepares conditioned `s` and `z`, then encodes atom-local features into token state, updates tokens with the diffusion transformer, and decodes token output back to atom-coordinate updates.

pair-bias boundary

Atom encoder: no pair bias. Token transformer: pair-biased attention.

`ESMFold2AtomEncoder.forward` has `z_ij` in the signature, but the forward path builds atom features, 3D RoPE parameters, and calls `SWAAtomTransformer` with `q_l`, `c_l`, and `attention_params` only.
